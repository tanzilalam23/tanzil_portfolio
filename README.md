## 🧩 How to Run

```bash
# 1. Clone the repo
git clone https://github.com/Jasleen-05/Web-Portfolio.git

# 2. Move into the directory
cd project

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
````
